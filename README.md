# Frank - Assignment Helper App

An intuitive, user-friendly web application designed to help students break down complex assignments into manageable steps. Perfect for students who struggle with task organization and need guidance without direct answers.

## Features

- **Intuitive Interface**: No instructions needed - just paste your assignment and get help
- **Smart Task Breakdown**: Automatically breaks assignments into smaller, manageable steps
- **Friendly Persona**: Relatable analogies and encouraging guidance
- **Interactive Checklists**: Track progress with checkboxes for each step
- **Local Storage**: All assignments are saved locally in your browser
- **Clean Design**: Monochromatic style with subtle color accents for maximum readability

## How to Use

1. **Open the App**: Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)

2. **Enter Your Assignment**: 
   - Paste your assignment text or type a question in the text box
   - Example: "I need to write a paper comparing themes from I am Malala to current US events"

3. **Get Help**: Click the "Get Help" button (or press Ctrl+Enter)

4. **Follow the Steps**: 
   - Read the friendly guidance message
   - Review the "How to Get Started" section
   - Work through each step and check off items as you complete them

5. **Save Your Work**: Click "Save Assignment" to keep it for later reference

6. **Access Saved Assignments**: View and reload any previously saved assignments from the bottom section

## Example Use Cases

- Compare/contrast essays
- Research papers
- Reading responses
- General assignment breakdown
- Any complex task that needs to be broken down

## Technical Details

- **Pure HTML/CSS/JavaScript**: No build tools or dependencies required
- **Local Storage**: All data saved in browser's localStorage
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **No Internet Required**: Works completely offline after initial load

## Getting Started
